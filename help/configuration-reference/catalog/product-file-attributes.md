# [!UICONTROL Catalog] > [!UICONTROL Product File Attributes]

## [!UICONTROL Configure Allowed File Types and Size]

![Attributi del file di prodotto](./assets/file-validation.png)

| Campo | [Ambito](../../getting-started/websites-stores-views.md#scope-settings) | Descrizione |
|--- |--- |--- |
| [!UICONTROL Allowed File Extensions] | Globale | Elenco separato da virgole dei tipi di file consentiti, ad esempio `pdf,doc,docx,txt`. Lascia vuoto per usare i tipi di file consentiti predefiniti: pdf, doc, docx, xls, xlsx, ppt, pptx, txt, csv, zip. |
| [!UICONTROL Maximum File Size] | Globale | Dimensione massima consentita del file in MB, ad esempio `20.0`. Il limite predefinito è 16 MB. La dimensione massima consentita del file è 31 MB. |
