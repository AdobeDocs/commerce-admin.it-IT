# Opzioni di configurazione del tipo di API USPS

>[!NOTE]
>
>Queste istruzioni includono i passaggi per selezionare il tipo di API USPS per l’integrazione: API di strumenti web o API REST. Le opzioni di configurazione del tipo di API sono disponibili solo se è stata applicata la patch di qualità [USPS REST API Migration](https://experienceleague.adobe.com/en/docs/commerce-operations/tools/quality-patches-tool/patches-available-in-qpt/v1-1-70/ac-15210)&#x200B;(AC-1520) all&#39;applicazione Commerce. Se non hai applicato la patch, il selettore del tipo di API non è disponibile e viene utilizzata l’API degli strumenti web USPS per impostazione predefinita.<br>Le API USPS sono il metodo preferito per l&#39;integrazione con USPS. L’API degli strumenti web USPS è obsoleta e potrebbe essere rimossa nelle versioni future.
